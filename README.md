# Postagram
A social media application where you can share your images with your friends.

## Features

- Create a post ,edit the post , delete the post , bookmark the post
- Like and comment feature on post
- Follow and unfollow the users
- Edit and update your Profile (profile-pic, about yourself)
- See the post of users whom you follow
